UrlDropper
==========


Drop you favorite feeds on a simple url list.

See a working example [here](http://etienneozeray.fr/UrlDropper).

## Install

Just change the line 
```php
$file = "http://yoursite.com/feeds.rss";
```
by your feeds url.
